    WhowatchPublish

    ふわっち自動枠取りブックマークレット

    これは何？

    ふわっちで配信する際、延長チケットを使わないと30分で配信が終わります。

    このブックマークレットは、自動で次の配信を始めるためのツールです。

    これを使えば「配信する」～「配信終了」「非公開で保存」～「配信する」～（以下同じ）が自動で行えます。

    動作環境(2020/4/22時点)

    • Google Chrome 81
    • OBS Studio

    使い方

    1. こちらをドラッグ＆ドロップ等でブックマークに登録
    2. ふわっちの「配信する」ページで「外部エンコーダーを使う」にする
    3. 登録したブックマークレットを実行
    4. 配信時間を入力して
    5. OBSで配信開始

    注意事項

    1. OBSの自動再接続機能が前提で作られています。
    2. 使用は 自己責任 でお願いします。
    3. ふわっち側の仕様変更があると動作しなくなります。
    4. 「配信ガイドラインに同意する」にチェックが入っていないと動作しません。
    5. 動作しない場合は@sugattorまでお知らせください。
